FAQs

Why can't I create and activity?

Only Strava users are allowed to create activities first log into your Strava account to create an activity. if you are already logged in to Strava you might have reached your limit of active activities. currently each user can only have 3 active activities at a time. If you need more please contact us.

I do not see any segments when I add a stage?

when adding stages <%=appName%> will only display starred segments. if you wish to add segments go to https://www.strava.com/athlete/segments/starred and add some starred segments.

How do I add a route to a activity

First go to Strava and create and save a route https://www.strava.com/athlete/routes. Create the activity and press the 'Set Route' button and select the route.

I can't join and event?

You must first log in using Strava or Facebook in order to join a activity

I submitted my activity to Strava but the results are not updated in the race

Results are only updated if you joined the race with logged in to Strava not Facebook. If they are still not updated after the activity is finished, and you have submitted your activity in Strava, you can press the 'Update Results' Button to force and updated
